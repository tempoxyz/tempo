# Tempo Partner Integrations

Build on Tempo with our ecosystem of infrastructure partners.

## RPC & Infrastructure

Connect to Tempo with enterprise-grade RPC providers:

- [**Tempo RPC**](/how-to/integrations/tempo-rpc) — Public endpoints and configuration
- [**QuickNode**](/how-to/integrations/quicknode) — Enterprise RPC with advanced features
- [**Tenderly**](/how-to/integrations/tenderly) — Debugging, simulation, and monitoring

## Wallets

Add wallet support to your Tempo application:

- [**MetaMask**](/how-to/integrations/metamask) — Add Tempo network to MetaMask
- [**Privy**](/how-to/integrations/privy) — Email, social login, and embedded wallets
- [**thirdweb**](/how-to/integrations/thirdweb) — All-in-one SDK with wallet UI
- [**Turnkey**](/how-to/integrations/turnkey) — Secure key infrastructure
- [**ZeroDev**](/how-to/integrations/zerodev) — Smart accounts and gas sponsorship

## Smart Contracts

Deploy and interact with contracts:

- [**Foundry**](/how-to/integrations/foundry) — Forge, Cast, and Anvil for Tempo
- [**Hardhat**](/how-to/integrations/hardhat) — Deploy with Hardhat
- [**thirdweb**](/how-to/integrations/thirdweb) — Pre-built contracts and deployment

## Interoperability

Bridge assets to and from Tempo:

- [**Bridge**](/how-to/integrations/bridge) — Stablecoin orchestration
- [**Bridging Guide**](/how-to/integrations/bridge) — Cross-chain transfers
