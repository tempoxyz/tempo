## Ecosystem Categories

| Category | Description |
|----------|-------------|
| **Issuance** | Create and manage stablecoins on Tempo with integrated minting, burning, and compliance controls. |
| **Wallets** | Secure custody solutions for users and institutions, from embedded wallets to self-custody options. |
| **Orchestration & Ramps** | On/off-ramp solutions and payment orchestration to move funds between traditional finance and Tempo. |
| **Infrastructure** | Node providers, RPCs, and core infrastructure to run and scale applications on Tempo. |
| **Data** | Indexers, explorers, and analytics tools to query blockchain data and build data-driven applications. |
| **Interoperability** | Bridges and cross-chain solutions to connect Tempo with other blockchains and ecosystems. |
| **Compliance & Fraud** | KYC/AML, transaction monitoring, and fraud prevention tools for regulatory compliance. |
| **Custody** | Institutional-grade custody solutions for securely managing digital assets on Tempo. |
