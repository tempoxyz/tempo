## Using Predeployed Contracts

### Interact with TIP-20 Tokens

```bash
# Get token symbol
cast erc20 symbol 0x20c0000000000000000000000000000000000001 \
  --rpc-url https://rpc.testnet.tempo.xyz

# Check balance
cast erc20 balance 0x20c0000000000000000000000000000000000001 \
  <YOUR_ADDRESS> \
  --rpc-url https://rpc.testnet.tempo.xyz
```

### Create a New Token

```bash
# Deploy a new TIP-20 token via factory
cast send 0x20c1000000000000000000000000000000000000 \
  "createToken(string,string,string,address,address)" \
  "My Token" "MTK" "USD" <QUOTE_TOKEN> <ADMIN> \
  --rpc-url https://rpc.testnet.tempo.xyz \
  --private-key <YOUR_PRIVATE_KEY>
```

### Use Multicall3

:::code-group

```typescript [TypeScript]
import { multicall3Abi } from 'viem'

```

