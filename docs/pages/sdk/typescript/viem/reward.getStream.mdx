# `reward.getStream`

Gets a reward stream by its ID.

## Usage

:::code-group

```ts twoslash [example.ts]
import { client, token } from './viem.config'

const stream = await client.reward.getStream({
  id: 1n,
  token,
})

console.log('Funder:', stream.funder)
// @log: Funder: 0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266
console.log('Total amount:', stream.amountTotal)
// @log: Total amount: 1000000000000000000000n
console.log('Rate per second:', stream.ratePerSecondScaled)
// @log: Rate per second: 385802469135802469135n
```

## Return Type

```ts
type ReturnValue = {
  /** Total amount allocated to the stream */
  amountTotal: bigint
  /** When the stream is scheduled to end (block timestamp) */
  endTime: bigint
  /** Address that funded the stream */
  funder: Address
  /** Per-second emission rate (scaled by ACC_PRECISION = 1e18) */
  ratePerSecondScaled: bigint
  /** When the stream started (block timestamp) */
  startTime: bigint
}
```

:::tip
If the stream has been canceled or doesn't exist, `funder` will be the zero address and all numeric fields will be `0`.
:::

## Parameters

### id

- **Type:** `bigint`

The stream ID to query. Note: `0` is never used for streams, only for immediate distributions.

### token

- **Type:** `Address`

Address of the TIP-20 token.
