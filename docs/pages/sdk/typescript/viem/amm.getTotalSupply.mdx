# `amm.getTotalSupply`

Gets the total supply of liquidity tokens for a specific pool.

## Usage

:::code-group

```ts twoslash [example.ts]
import { client } from './viem.config'

const totalSupply = await client.amm.getTotalSupply({
  userToken: '0x20c0000000000000000000000000000000000000',
  validatorToken: '0x20c0000000000000000000000000000000000001',
})

console.log('Total liquidity supply:', totalSupply)
// @log: Total liquidity supply: 1000000000000000000000n
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
// [!include ~/snippets/viem.config.ts:setup]
```

:::

## Return Type

```ts
type ReturnType = bigint // Total liquidity supply
```

## Parameters

### userToken

- **Type:** `Address | bigint`

Address or ID of the user token.

### validatorToken

- **Type:** `Address | bigint`

Address or ID of the validator token.
