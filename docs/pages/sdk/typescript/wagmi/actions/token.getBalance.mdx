# `token.getBalance`

Gets the token balance of an address.

## Usage

:::code-group

```ts twoslash [example.ts]
import { Actions } from 'tempo.ts/wagmi'
import { config } from './wagmi.config'

const balance = await Actions.token.getBalance(config, {
  account: '0x742d35Cc6634C0532925a3b844Bc9e7595f0bEbb',
  token: '0x20c0000000000000000000000000000000000000',
})

console.log('Balance:', balance)
// @log: Balance: 10500000n
```

```ts twoslash [wagmi.config.ts] filename="wagmi.config.ts"
// @noErrors
// [!include ~/snippets/wagmi.config.ts:setup]
```

:::

## Return Type

```ts
type ReturnType = bigint // Balance amount
```

## Parameters

### account

- **Type:** `Account | Address`

Account to check balance for.

### token

- **Type:** `Address | bigint`

Address or ID of the TIP-20 token.
