# `token.watchApprove`

Watches for token approval events on TIP20 tokens.  

## Usage

:::code-group

```ts twoslash [example.ts]
import { Actions } from 'tempo.ts/wagmi'
import { config } from './wagmi.config'

const unwatch = Actions.token.watchApproveSync(config, {
  onApproval: (args, log) => {
    console.log('Amount:', args.amount)
    console.log('Owner:', args.owner)
    console.log('Spender:', args.spender)
  },
})

// Later, stop watching
unwatch()
```

```ts twoslash [wagmi.config.ts] filename="wagmi.config.ts"
// @noErrors
// [!include ~/snippets/wagmi.config.ts:setup]
```

:::

## Return Type

```ts
type ReturnType = () => void
```

Returns a function to unsubscribe from the event.

## Parameters

### onApproval

- **Type:**

```ts
declare function onApproval(args: Args, log: Log): void

type Args = {
  /** Amount approved */
  amount: bigint
  /** Address of the token owner */
  owner: Address
  /** Address of the spender */
  spender: Address
}
```

Callback to invoke when tokens are approved.

### token

- **Type:** `Address | bigint`

Address or ID of the TIP20 token to watch.
