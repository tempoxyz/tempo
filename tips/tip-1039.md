# TIP-1039: Key Authorization Scope Gas Surcharge

## Abstract

TIP-1011 introduced call-scope storage-slot charges for key authorizations at T3, but those charges only cover the dynamic SSTORE rows. The surrounding bookkeeping — clearing the empty scope tree, maintaining target and selector sets, and writing recipient sets — remains unpriced. This TIP adds a rounded intrinsic gas surcharge at T4 to close that gap.

## Motivation

T3 key authorizations undercharge intrinsic gas for the call-scope helper path. The existing intrinsic calculation charges the dynamic storage rows for the persisted scope tree, but does not account for the surrounding bookkeeping in `replace_allowed_calls`: clearing the empty tree for a fresh key, maintaining target and selector sets, and writing recipient sets. This makes unrestricted and small-scoped authorizations underpriced, with the gap growing as scope trees get larger.

---

# Specification

Implementations MUST charge an additional rounded surcharge on top of the existing TIP-1011 scope-slot charge for every key authorization that includes call-scope configuration.

The surcharge uses the same scope cardinalities as TIP-1011 (`S` = number of target scopes, `K` = number of selector rules, `W` = number of recipients):

```text
extra_scope_gas = 5_000 + 7_000*S + 7_000*K + 5_000*W
```

Where:
- **5,000 base** covers the always-run empty-tree clear and restricted/unrestricted mode bookkeeping.
- **7,000 per target** covers the set-maintenance work around each target scope layer.
- **7,000 per selector** covers the set-maintenance work around each selector rule.
- **5,000 per recipient** covers the extra recipient-set persistence.

For unrestricted authorizations (no `allowed_calls`), only the 5,000 base is charged.

The surcharge intentionally rounds upward. The design goal is to avoid materially underpricing larger scope trees while keeping pricing simple and predictable; slight overcharging is acceptable.

# Invariants

1. The surcharge MUST be applied in addition to, not instead of, the TIP-1011 scope-slot SSTORE charges.
2. The surcharge MUST only be active at T4+. Pre-T4 blocks MUST NOT include it.
3. Unrestricted key authorizations MUST still pay the 5,000 base surcharge.
